# House Price Prediction Web App

This is my Machine Learning mini project where I built a house price prediction system and deployed it as a web application using Streamlit.

The model is trained using Random Forest Regression and can predict house prices based on user input features.

## Live App

You can try the web app here:
https://hpp-99x.streamlit.app/

## What I Did

- Performed data preprocessing
- Selected relevant features
- Trained a Random Forest regression model
- Evaluated model performance using R² score
- Built a Streamlit web interface
- Deployed the app using Streamlit Cloud

## Model Details

- Algorithm: Random Forest Regressor
- Type: Supervised Learning (Regression)
- Performance Metric: R² Score ≈ 0.75
- Libraries Used:
  - scikit-learn
  - pandas
  - numpy
  - joblib

## Tech Used

- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Streamlit  

## How to Run Locally

1. Clone the repository:
   git clone https://github.com/raazan99x-netizen/ML_Mini_Project.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   streamlit run app.py
